# 安装Jenkins
操作系统：Centos7

[官网地址](https://jenkins.io/)

[下载地址](https://pkg.jenkins.io/redhat/)

## 解决依赖环境
```
yum install java-1.8.0-openjdk.x86_64 -y
```

## 安装
```
rpm -ivh jenkins-2.147-1.1.noarch.rpm

# 启动
service jenkins start

# 访问
http://192.168.199.5:8080
```
