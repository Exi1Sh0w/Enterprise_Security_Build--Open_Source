### 10大注意事项
* 浏览器拦截混合访问请求

* 前端HTML、JS资源引用存在HTTP

* 移动端适配HTTPS

* 项目中存在的配置问题

* 关于request.getScheme()的坑

* SSL证书类型

* Nginx配置同一个Server段不支持配置多个证书

* Nginx配置HTTP强制跳转HTTPS

* 所有环节均要进行升级
